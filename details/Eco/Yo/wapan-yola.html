Ile Ajumọ Asigbe Agbaye Ará Ile

Nipa WAPAN

The Waste Pickers Association of Nigeria (WAPAN) jẹ ile ajumọ ará ile ti o lo ara ninu asigbe agbaye ati agbaye ni Yola. WAPAN n lo ara lori gbigbẹ ẹkó, sisagbe ipo iṣẹ, ati sisagbe iṣẹ fun awon alabọde.

Ilojana Ilosiwaju

  • Agbẹkọ para ipo ati ipo iṣẹ ti awon olugbe ẹkó
  • Asigbe agbaye nipa sisagbe ẹkó
  • Agbaye ará ile nipa awon ere ẹkó
  • Sisọ aje para awon olugbe ẹkó
  • Atilẹyin para sisagbe imulẹ ẹkó alailẹka

Awon Iṣe Atilẹyin Alabọde

  • Awon Iṣẹ Ikẹkọ: Sisagbe iṣẹ para sisagbe ẹkó
  • Idiwọ Aje: Idiwọ si awon alabọde para nkan iye
  • Ero Alaibalẹ: Ṣiṣo awon ero alabọde
  • Atilẹyin Iju: Awon iṣẹ ilera ati awon iṣẹ ijoba miiran
  • Agbẹkọ: Sisagbe ipo awon alabọde

Awon Ifowosowopo Agbaye

  • Gbigbe ati ṣiṣẹ awon nkan iye lati ara ẹkó
  • Ṣiṣi atiwọ aaye ẹkó ti o de si ibadandun
  • Ṣiṣi awon iṣẹ yiyipada para ará ile
  • Ṣiṣi agbaye nipa imulẹ ẹkó
  • Atilẹyin para sisagbe aje alailẹka

Ipa Ile-Iṣe Ati Ijoba

  • Ṣiṣi aje para awon igba ti awon olugbe ẹkó
  • Ṣiṣi imulẹ iṣẹ ni imulẹ ẹkó
  • Sisọ awon aye iṣẹ para awon ará ile alailẹka
  • Atilẹyin para aye ile
  • Atilẹyin para ṣiṣi atiwọ ilẹ ẹkó
